# Digital Logic Adventure

Digital Logic Adventure is an interactive Python program that introduces basic digital logic concepts through a simulated voltage transmission signal.

The program generates voltage data, lets the user clean up indeterminate values, converts the signal into a binary bitstream, visualizes the bitstream as an image, demonstrates logic gates, and shows how XOR can scramble and unscramble binary data.

## Project Overview

This project was created as a final Python programming project. It combines programming and digital logic concepts.

The program walks the user through:

* Generating a bimodal gaussian distribution of voltages
* Converting voltage levels into binary data
* Visualizing binary data as a pixel image
* Applying AND, OR, and XOR logic gates
* Displaying a timing diagram
* Using XOR to scramble and unscramble a bitstream

## Concepts Demonstrated

This project demonstrates:

* Python lists and NumPy arrays
* Random number generation
* Boolean masks
* Conditional logic
* Loops and input validation
* Bitwise logic operations
* Error handling
* Matplotlib visualizations
* Digital logic concepts including HIGH, LOW, AND, OR, and XOR

## Requirements

This project uses:

* Python 3
* NumPy
* Matplotlib

## Setting Up a Virtual Environment

A virtual environment is not included in this repository. To create one locally, run:

```bash
python3 -m venv venv
```

Activate the virtual environment:

```bash
source venv/bin/activate
```

Install the required packages:

```bash
pip install numpy matplotlib
```

## How to Run

After installing the required packages, run:

```bash
python Digical_Logic_Final_project.py
```
